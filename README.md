# Pulso - Asistente Virtual para la Salud

Proyecto para el Hackathon Nicaragua 2026.

## Objetivo del Proyecto
Reducir la desinformación y las brechas de acceso a la salud, brindando orientación primaria contextualizada y accesible a familias nicaragüenses.

## Solución Integral
| Funcionalidad | Impacto |
| :--- | :--- |
| Orientación Médica | IA validada sobre signos/síntomas |
| Monitoreo Completo | Antecedentes, Citas, Vacunas |
| Alertas Inteligentes | Alertas epidemiológicas en tiempo real |
| Localizador | Mapa de centros de salud cercanos |

## Tecnologías
- Backend: Go (Gin, GORM, SQLite)
- Frontend: Astro + Tailwind CSS
- IA: Integración API para Asistente Virtual

## Estructura del Monorepo
```text
/
├── backend/            # API Go + Lógica
└── frontend/           # App Web (Astro)
```
